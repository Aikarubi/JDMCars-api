Introducción

Bienvenido a la API JDM, una plataforma diseñada para los amantes de los coches japoneses. Aquí encontrarás información detallada sobre distintos modelos, especificaciones técnicas y características clave.

Esta guía te ayudará a entender cómo utilizar la API de forma efectiva y aprovechar sus funciones al máximo.

Ruta Principal

La API parte de la ruta principal proporcionada por Render: https://jdm-api.onrender.com/v1/cars. A partir de esta base, puedes acceder a todos los endpoints disponibles.

Paginación

La API utiliza un sistema de paginación para facilitar la navegación entre grandes volúmenes de datos. Puedes utilizar los parámetros ?page y ?limit para controlar los resultados.

Ejemplo de input:

GET https://api-jdm.onrender.com/v1/cars?page=1&limit=2

Estructura y Endpoints

La API está organizada en rutas claras y sencillas, comenzando por la base /v1/cars. Desde aquí, puedes acceder a diversos endpoints para obtener listas de coches, detalles específicos, estadísticas y realizar búsquedas avanzadas según tus necesidades.

  • /v1/cars o /v1/cars/: Obtiene todos los coches almacenados (getAllCars).
  • /v1/cars/:id: Obtiene la información detallada de un coche específico mediante su ID (getOneCar).
  • /v1/cars/random: Devuelve un coche aleatorio (getRandomCar).
  • /v1/cars/brands: Muestra todas las marcas disponibles (getAllBrands).
  • /v1/cars/stats: Muestra la media de todas las estadísticas de los coches (getGlobalStat).

Ejemplo básico de uso: https://api-jdm.onrender.com/v1/cars?brand=Mazda para filtrar por marca.

Parámetros y Filtrado

La API permite utilizar múltiples parámetros para filtrar los resultados y encontrar modelos específicos de forma sencilla. Puedes combinar varios filtros en la misma URL para afinar las búsquedas según lo que necesites. A continuación, se muestran algunos ejemplos:

  • /v1/cars?brand=Nissan: Filtra por marca.
  • /v1/cars?model=Supra: Filtra por modelo.
  • /v1/cars?horsepower=276: Filtra por potencia en caballos.
  • /v1/cars?engine=2JZ-GTE: Filtra por tipo de motor.
  • /v1/cars?country=Japan: Filtra por país de origen.
  • /v1/cars?transmission=Manual: Filtra por tipo de transmisión.
  • /v1/cars?drivetrain=RWD: Filtra por tipo de tracción.
  • /v1/cars?max_speed=250: Filtra por velocidad máxima.
  • /v1/cars?acceleration=5.0: Filtra por aceleración.
  • /v1/cars?weight=1400: Filtra por peso.
  • /v1/cars?price=25000: Filtra por precio.
  • /v1/cars?fuel_consumption=10.0%20L/100%20km: Filtra por consumo de combustible.
  • /v1/cars?yearStart=1995&yearEnd=2000: Filtra por rango de años.

Además, es posible combinar múltiples parámetros para obtener resultados más precisos. Por ejemplo:

https://api-jdm.onrender.com/v1/cars?brand=Subaru&engine=EJ20%20Turbo&transmission=Manual

Objetivos y Mejoras Futuras

Entre los objetivos a futuro, se planea añadir motocicletas a la API y desarrollar funciones avanzadas como comparaciones detalladas entre vehículos y estadísticas personalizadas.

La documentación también será ampliada, incluyendo ejemplos prácticos y casos de uso detallados para simplificar la implementación por parte de los desarrolladores.
