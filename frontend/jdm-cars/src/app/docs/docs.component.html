Introducción

Bienvenido a la API JDM, una plataforma diseñada para los amantes de los coches japoneses. Aquí encontrarás información detallada sobre distintos modelos, especificaciones técnicas y características clave.

Esta guía te ayudará a entender cómo utilizar la API de forma efectiva y aprovechar sus funciones al máximo.

Ruta Principal

La API parte de la ruta principal proporcionada por Render: https://api-jdm.onrender.com/v1/cars. A partir de esta base, puedes acceder a todos los endpoints disponibles.

Paginación

La API utiliza un sistema de paginación para facilitar la navegación entre grandes volúmenes de datos. Puedes utilizar los parámetros ?page y ?limit para controlar los resultados.

Ejemplo de input:

GET https://api-jdm.onrender.com/v1/cars?page=1&limit=2

Ejemplo de output:

    "page": 1,
    "limit": 2,

Estructura y Endpoints

La API está organizada en rutas claras y sencillas, comenzando por la base /v1/cars. A partir de ahí, puedes acceder a diferentes endpoints para obtener listas de coches, detalles específicos y realizar búsquedas avanzadas.

Por ejemplo, puedes utilizar /v1/cars?brand=Nissan para filtrar por marca.

Parámetros y Filtrado

La API permite utilizar parámetros como marca, año y tipo de motor para filtrar los resultados según tus necesidades. Esto facilita encontrar modelos específicos sin complicaciones.

Ejemplo de uso: /v1/cars?year=1995&engine=turbo.

Objetivos y Mejoras Futuras

Entre los objetivos a futuro, se planea añadir motocicletas a la API, implementar nuevas funciones como comparaciones detalladas y mejorar los sistemas de filtrado.

Además, se trabajará en la optimización del rendimiento, asegurando tiempos de respuesta rápidos y eficientes incluso con grandes volúmenes de datos.

La documentación también será ampliada, incluyendo ejemplos prácticos y casos de uso detallados para simplificar la implementación por parte de los desarrolladores.
